쌤 안녕하세요 홍정환입니다. [Advanced 프로젝트] 관련해서 쌤께서 예시로 말씀해 주신 것으로 진행을 하려고 합니다.
- 강점 : 거의 대부분 말씀해주셨지만, 전자제품중 이미지넷 등에 클래스가 존재하고 또 인터넷에서 이미지를 쉽게 구할 수 있는 마우스로 특정해서 진행해 보고자 합니다.
- 약점 : 조립이 덜된 상태의 마우스 이미지를 구할 수 있을지, 못구한다면 완성된 상태의 이미지만으로 학습해도 원하는 결과를 얻을 수 있을지 모르겠습니다.
아래 기능부분은 실제 구현 가능한지 불가능한지 여부는 아직 알수 없지만, 제시해주신 기능정의를 이용하여 제가 상상할 수 있는 부분을 아래와 같이 구체화하여 일단 기술해보았습니다. 

---------------------------------------------   아          래 ---------------------------------------------

- 제목 : 마우스 조립/테스트라인에서 사용할 딥러닝을 이용한 품질관리 시스템

- 기능 : 마우스 사진파일 입력을 기본으로하고 

1. 입력된 사진파일을 완제품/반제품인지 분류하는 기능
   - 완제품 : 모든 조립이 완전하게 이루어진 상태
   - 반제품 : 필요 부품중 하나라도 조립이 이루어지지 않은 상태

2. 입력된 사진파일에 메타정보를 추가(addon)하는 기능
   - Addon 항목 : 완제품/반제품여부, 반제품일 경우 누락부품

3. 입력된 사진파일의 주요 객체의 특성을 파악하는 기능
   - 주요 객체의 특성 : 부품명

- 모델 : CNN
- 학습데이터 : ImageNet, AI Hub의 이미지 활용
